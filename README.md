# 📚 Book Store Management System

A console-based Book Store Management application built in **C++** using file handling. This was my Programming Fundamentals project at the University of Central Punjab.

## 🚀 Features

- ➕ Add new book records
- 📋 Display all books
- 🔍 Search book by **Name**
- 🔍 Search book by **Genre**
- ✏️ Update existing book record
- 🗑️ Delete book record by ID

## 🛠️ Technologies Used

- **Language:** C++
- **Concepts:** File Handling (fstream, ofstream, ifstream), Functions, Loops, String manipulation
- **Storage:** CSV-format flat file (`bookstore.txt`)

## 📁 How to Run

1. Open the project in any C++ IDE (Dev-C++, VS Code, Code::Blocks)
2. Compile `project_pf.cpp`
3. Run the executable
4. Use the menu to add, search, update or delete book records

## 📂 File Structure

```
book-store-management/
│
├── project_pf.cpp       # Main source code
└── bookstore.txt        # Auto-generated data file
```
